# Подзадача P16.6: Реализация CLI интерфейса

## Описание
Реализовать CLI команды для интерактивной работы с LLM wrapper, включая поддержку режимов plan/act как в Cline

## Приоритет
HIGH

## Технические детали

### 1. CLI команды
Источник: https://github.com/cline/cline/blob/main/src/shared/api.ts

- [ ] Добавить команду `cline` для интерактивной работы
- [ ] Реализовать команду `--provider` для выбора провайдера
- [ ] Реализовать команду `--model` для выбора модели
- [ ] Добавить команды для режимов `--plan` и `--act`
- [ ] Добавить команду `--settings` для настройки провайдеров

### 2. Интерактивный режим
- [ ] Реализовать интерактивный чат с LLM
- [ ] Добавить поддержку истории сообщений
- [ ] Реализовать команды для переключения моделей в процессе сессии
- [ ] Добавить команды для просмотра токен-использования

### 3. Система настройки
- [ ] Реализовать систему хранения настроек провайдеров
- [ ] Добавить команды для сохранения/загрузки профилей
- [ ] Реализовать валидацию API ключей
- [ ] Добавить команды для переключения между профилями

### 4. Интеграция с существующим CLI
- [ ] Интегрировать с текущим `cli.py` в HordeForge
- [ ] Добавить подкоманды для LLM функций
- [ ] Обеспечить совместимость с существующими командами

## Критерии готовности (DoD)
- [ ] CLI команды работают корректно
- [ ] Режимы plan/act поддерживаются
- [ ] Интерактивный чат функционирует
- [ ] Система настройки работает
- [ ] Тесты покрывают CLI функции
- [ ] Документация обновлена

## Риски
- Конфликты с существующими командами CLI
- Сложность интеграции интерактивного режима
- Совместимость с разными операционными системами

## Комментарии
Важно обеспечить удобство использования CLI интерфейса с возможностью быстрого переключения между провайдерами и моделями.
