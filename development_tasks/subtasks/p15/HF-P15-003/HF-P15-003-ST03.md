## Тест-дизайн

```python
def test_benchmark_comparison():
    # Arrange
    test_issues = [
        {"title": "Fix auth bug", "body": "..."},
        {"title": "Add logging", "body": "..."},
        # ... 18 more
    ]
    
    # Act
    baseline_results = run_benchmark("feature_pipeline", test_issues, use_memory=False)
    memory_results = run_benchmark("feature_pipeline", test_issues, use_memory=True)
    
    # Assert
    assert baseline_results.success_rate < memory_results.success_rate
    improvement = (memory_results.success_rate - baseline_results.success_rate) / baseline_results.success_rate
    assert improvement >= 0.30  # 30% improvement
    
    assert baseline_results.avg_prompt_size / memory_results.avg_prompt_size >= 5
```
