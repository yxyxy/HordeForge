# Подзадача P17.3: Пакетная обработка эмбеддингов

## Описание
Необходимо модифицировать `IngestionPipeline` для поддержки пакетной обработки эмбеддингов с целью ускорения процесса вычисления векторов.

## Цели
1. Модифицировать `IngestionPipeline` для поддержки пакетной обработки
2. Реализовать буферизацию текстов для вычисления эмбеддингов пакетами
3. Оптимизировать размеры батчей для максимальной производительности
4. Обеспечить нормализацию векторов в пакетном режиме

## Технические требования

### 1. Пакетная обработка
- Реализовать буферизацию текстов для накопления батчей перед вычислением эмбеддингов
- Обеспечить возможность настройки размера батча
- Поддерживать обработку батчей разного размера (включая последний неполный батч)

### 2. Оптимизация производительности
- Максимизировать размер батча в пределах доступной памяти
- Обеспечить эффективное использование GPU/CPU при вычислении
- Минимизировать количество обращений к модели

### 3. Нормализация векторов
- Выполнять нормализацию векторов в пакетном режиме
- Обеспечить корректную нормализацию для всех векторов
- Поддерживать различные методы нормализации (L2, L1 и др.)

## Подробное описание

### 1. Буферизация текстов (TextBuffer)
Класс `TextBuffer`:
- Накапливает тексты для вычисления эмбеддингов
- Имеет настраиваемый размер батча
- Автоматически выдает батчи при достижении порога
- Поддерживает выдачу остаточного батча при завершении

### 2. Пакетный вычислитель эмбеддингов (BatchEmbedder)
Класс `BatchEmbedder`:
- Принимает батчи текстов для вычисления эмбеддингов
- Использует `TextEmbedding` из fastembed для пакетного вычисления
- Выполняет нормализацию векторов в пакетном режиме
- Возвращает пакеты векторов с сохранением порядка

### 3. Модификация IngestionPipeline
Обновить `rag/ingestion.py`:
- Добавить поддержку пакетной обработки в `IngestionPipeline`
- Интегрировать `TextBuffer` и `BatchEmbedder`
- Обеспечить настройку размера батча через параметры
- Обновить логику обработки текстов для использования батчей

### 4. Оптимизация параметров
- Определить оптимальный размер батча для различных сценариев
- Реализовать автоматическую настройку размера батча
- Обеспечить возможность ручной настройки пользователем
